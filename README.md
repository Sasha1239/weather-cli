# Weather в терминале


Как запускать:

```bash
$ npm install
```

```bash
$ node weather.js --help
Usage: weather [options]

Выводит текст hello world в терминал

Options:
  -V, --version  output the version number
  --echo         
  -h, --help     output usage information


$ node weather.js --echo "test"
test
```
